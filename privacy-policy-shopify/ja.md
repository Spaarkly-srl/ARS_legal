Shopify 申請用ドラフト版

本プライバシーポリシーは、**Spaarkly s.r.l.**（以下「Spaarkly」、「ARShades」、「当社」といいます）が **ARShades VTO for Shopify** アプリ（以下「Shopify アプリ」または「本アプリ」といいます）に関連して個人データその他の情報をどのように取り扱うかを説明するものである。

本ポリシーは Shopify アプリに固有のものであり、Shopify に関連して取り扱われるデータ（マーチャント、ストア、商品、設定および帰属に関するデータ）に限定される。本アプリを通じて起動される場合がある ARShades のエンドユーザー向けサービス（Virtual Try-On、3D/AR Viewer および AR PD Meter など）が行う取扱いは、本ポリシーの対象外であり、**ARShades コアプライバシーポリシー**（バージョン5.3）によって規律される。同ポリシーは [https://github.com/Spaarkly-srl/ARS_legal](https://github.com/Spaarkly-srl/ARS_legal/blob/main/privacy-policy/ja.md) にて入手可能であり、ARShades エクスペリエンス内でショッパーに対して表示される。

## 2. Shopify アプリの機能

Shopify アプリは、マーチャントの Shopify ストアを ARShades プラットフォームに接続する。本アプリにより、マーチャントは以下のことが可能となる。

* ARShades ライセンストークンを Shopify ストアに接続すること。
* ARShades カタログを作成または接続すること。
* SKU、EAN その他類似の商品識別子を用いて、Shopify の商品およびバリエーションを ARShades の 3D/VTO アセットと照合すること。
* Virtual Try-On、3D/AR Viewer および AR PD Meter のための theme blocks を追加すること。
* 色、ボタンスタイル、モーダル設定および一部のブランディングアセットをカスタマイズすること。
* VTO/3D の帰属およびコンバージョン分析のために Shopify Web Pixel イベントを利用すること。

本アプリは Shopify 上で無償でインストールできる。基盤となる ARShades サービスを利用するには、Shopify 外で購入する別途の ARShades サブスクリプションが必要となる場合がある。

## 3. 当社が取り扱うデータの類型

### 3.1 マーチャントおよび Shopify Admin データ

マーチャントが本アプリをインストールまたは利用する際、当社は以下のデータを取り扱う場合がある。

* Shopify のショップドメインおよびストア URL。
* Shopify OAuth セッションデータ。
* アクセストークンおよび認可されたスコープ。
* Shopify の管理ユーザー識別子、ならびに Shopify から提供される場合には、姓、名、メールアドレス、ロケール、コラボレーターの状態、アカウント所有者の状態およびメール認証の状態。
* 本アプリのインストール、設定および利用の状態。

これらのデータは、マーチャントの認証、埋め込み型の本アプリの運用、本アプリのセッションの維持、ならびにサポートおよびセキュリティの提供のために利用される。

### 3.2 ARShades ライセンス、カタログおよび設定データ

当社は、Shopify ストアを ARShades に接続するために必要な情報を取り扱う。これには以下が含まれる。

* ARShades ライセンストークンまたはサブスクリプション識別子。
* ARShades カタログ ID、カタログ名およびカタログの状態。
* 3D viewer ID、ならびに VTO や AR PD Meter の提供可否といった ARShades サービスの機能。
* Shopify metafield として保存される本アプリの設定（`licenseId`、`catalogueId`、`viewerId`、`webDomain`、`studioDomain`、`has_vto` および `has_arpd` を含む）。
* 色、モーダル設定およびボタン設定などのスタイル設定。
* ARShades の許可画面またはポリシー画面で使用されるロゴなど、マーチャントがアップロードしたブランディングアセット。

### 3.3 Shopify 商品およびバリエーションデータ

商品を ARShades アセットと照合し、利用可能な場合にのみ VTO/3D 機能を表示するため、本アプリは以下のデータを読み取り、取り扱う場合がある。

* Shopify の商品 ID、タイトル、ハンドルおよび注目画像 URL。
* Shopify のバリエーション ID。
* SKU、barcode、EAN または UPC。
* 既存の ARShades metafield の値。
* ARShades カタログの商品識別子およびバリエーション識別子。

本アプリは、`catalogueVariantId` および `catalogueProductId` を含む ARShades の技術的な metafield を Shopify のバリエーションに書き込む場合がある。これらの metafield は、ある商品またはバリエーションが ARShades VTO または 3D/AR Viewer を起動できるか否かを判定するために、ストアフロントの theme blocks によって使用される。

### 3.4 本アプリが取り扱うストアフロントデータ

ショッパーがマーチャントのストアフロント上で VTO または 3D/AR Viewer を利用する際、本アプリは以下の技術的かつ非プロファイルのデータを取り扱う場合がある。

* 商品識別子およびバリエーション識別子。
* ARShades カタログ識別子。
* 選択された、または最後に試したバリエーションの識別子。
* デバイス、ブラウザ、言語およびセッションに関する情報。
* VTO または 3D セッションが開始されたことを示す技術的イベント。

これらのデータは、ストアフロント上で適切なエクスペリエンスを表示するため、および第3.6条に記載する帰属分析を実現するために利用される。

### 3.5 ARShades エンドユーザーエクスペリエンス（VTO、3D/AR Viewer、AR PD Meter）

ストアフロントから起動される ARShades エクスペリエンス内で行われるカメラベースの処理（Virtual Try-On のレンダリングおよび AR PD Meter による瞳孔間距離の計算を含む）は、Shopify アプリではなく ARShades サービスによって実行され、ARShades コアプライバシーポリシーによって規律される。

同ポリシーに記載のとおり、要約すると次のとおりである。すなわち、カメラの画像および映像ストリームは保存されず、顔テンプレートや生体識別子は作成されない。また、AR PD Meter については、ショッパーを識別しないカメラ由来の技術データが、欧州連合内に所在する ARShades バックエンドへ一時的に送信され、測定結果を返すためにのみ使用され、処理後は破棄される場合がある。Shopify アプリ自体は、カメラデータにアクセスせず、これを保存または受領することもない。

### 3.6 Shopify Web Pixel およびコンバージョン分析

本アプリは、VTO/3D の帰属およびコンバージョンのパフォーマンスを測定するために Shopify Web Pixel を利用する。当該ピクセルは以下を取り扱う場合がある。

* ショップドメイン。
* イベントの種類およびタイムスタンプ。
* 仮名化された VTO/3D セッション ID。
* 商品識別子およびバリエーション識別子。
* ARShades カタログの商品識別子およびバリエーション識別子。
* カート追加の帰属のためのカート識別子。
* 購入の帰属のためのチェックアウト／注文の識別子、合計金額、通貨およびラインアイテムのメタデータ。
* 比較分析のため、購入が VTO/3D セッションの後に行われたか、または VTO/3D セッションなしで行われたか。

当該ピクセルは、氏名、メールアドレス、電話番号または配送先住所といった直接的なショッパーのプロファイルデータを ARShades 分析バックエンドへ送信しないように設計されている。

ストアフロントのピクセルは、帰属のためにアクティブな VTO/3D セッションを記憶する目的で、ブラウザのローカルストレージを使用する場合がある。このローカルセッションデータは、7日後に失効するか、または追跡された購入の後に消去される。

当該ピクセルは、Shopify の Customer Privacy API およびマーチャントの同意設定に従い、訪問者の同意が本アプリの宣言する同意カテゴリー（`analytics` など）と一致する場合にのみ、Shopify のピクセルマネージャーによって読み込まれる。当社はこれらの同意シグナルを尊重し、必要な同意なく収集されたピクセルイベントを取り扱わない。

## 5. データの保持

当社は、本ポリシーに記載する目的のために必要な期間に限りデータを保持する。

* アクティブな本アプリ設定、Shopify metafield、商品／バリエーションのマッピングおよび OAuth セッションは、本アプリがインストールされている間、かつサービスの運用に当該データが必要である間、保持される。
* ストア固有の本アプリデータは、Shopify が有効な `shop/redact` コンプライアンス Webhook を送信した場合、法令上または正当なセキュリティ目的のために保持が要求される場合を除き、削除または匿名化される。
* ピクセル分析、コンバージョンイベント、技術ログ、セッションデータおよび類似の履歴データは、最長14か月間保持され、その後、集計または匿名化される。
* マーチャントがアップロードしたブランディングアセットおよびスタイル設定は、アクティブなサービスのために必要である間保持され、削除、アンインストールまたは適用される保持期間の満了の後、除去または匿名化される。
* ARShades エンドユーザーエクスペリエンス内で取り扱われるデータ（AR PD Meter を含む）は、ARShades コアプライバシーポリシーに記載のとおり保持される。

## 6. Shopify のプライバシーおよびコンプライアンス Webhook

本アプリは、以下のための Shopify プライバシーおよびコンプライアンス Webhook のエンドポイントを実装している。

* `customers/data_request`。
* `customers/redact`。
* `shop/redact`。

Webhook リクエストは Shopify の HMAC 署名を用いて検証され、無効な署名を伴うリクエストは拒否される。当社は、Webhook を受領してから30日以内にデータ請求および削除の措置を完了する。

本アプリのデータベースは、ショッパーの氏名、メールアドレス、電話番号または配送先住所といった直接的なエンドカスタマーのプロファイルデータを保存しない。顧客レベルの請求については、当社は Shopify の Webhook を検証し、本アプリが保有するデータに応じて対応する。

`shop/redact` は、本アプリがアンインストールされてから48時間後に Shopify によって送信される。当社は、これを受領した時点で、法令上またはセキュリティ上の保持要件に従うことを条件として、Shopify セッション、同期済みのバリエーションマッピングおよびショップのピクセル設定を含む、当該ショップに保存された本アプリデータを削除する。

## 10. ショッパーのカメラ許可

カメラへのアクセスは、ARShades エクスペリエンス（VTO、AR PD Meter）内でのみ、ショッパーのブラウザまたはデバイスのオペレーティングシステムによって要求される。ショッパーは、ブラウザまたはデバイスの設定を通じてカメラアクセスを拒否または取り消すことができる。拒否された場合、当該エクスペリエンスは機能しないことがある。Shopify アプリ自体はカメラにアクセスしない。

ARShades エクスペリエンス内でカメラデータがどのように取り扱われるかについては、ARShades コアプライバシーポリシーに記載されている。
